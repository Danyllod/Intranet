# Conexão CRESM - Portal de Notícias

Um portal profissional de notícias desenvolvido em Flask com design responsivo.

## 📋 Características

- Design responsivo e profissional
- Navegação intuitiva com múltiplas categorias
- Seções destacadas e tendências
- Sistema completo de notícias
- Footer com redes sociais

## 🚀 Instalação

1. Criar ambiente virtual:
   ```bash
   python -m venv venv
   ```

2. Ativar ambiente:
   ```bash
   venv\Scripts\activate
   ```

3. Instalar dependências:
   ```bash
   pip install -r requirements.txt
   ```

4. Executar:
   ```bash
   python app.py
   ```

Acesse: http://127.0.0.1:5000
